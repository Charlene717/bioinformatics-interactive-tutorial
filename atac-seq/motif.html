Motif 與 Footprinting

從開放區域進一步推斷「是哪些轉錄因子在這裡結合」。

From open regions to inferring which transcription factors are bound there.

兩個層次

Motif Enrichment vs Footprint

🔤

Motif Enrichment

問：「我的 peaks 是否富集某個 TF 的 motif (DNA 序列模式)？」用 HOMER、MEME-ChIP、AME 等掃 motif 資料庫 (JASPAR、CIS-BP)，再做超幾何檢定。

"Are my peaks enriched for a TF's motif (DNA pattern)?" Scan motif DBs (JASPAR, CIS-BP) with HOMER, MEME-ChIP, AME, then hypergeometric test.

👣

Footprinting

問：「motif 上方是否真的有 TF 蛋白蹲坐？」TF 結合會在 Tn5 cut 信號上留下「下凹」——蛋白擋住 Tn5。工具：HINT-ATAC、TOBIAS、Wellington。

"Is a TF actually bound at this motif?" A bound TF blocks Tn5 → leaves a "dip" in the cut-site signal. Tools: HINT-ATAC, TOBIAS, Wellington.

程式碼

實作範例

# Motif enrichment in your peaks vs background
findMotifsGenome.pl sample1.narrowPeak hg38 motif_out/ \
  -size 200 -mask -p 8
# 結果在 knownResults.html 與 homerResults.html
library(chromVAR); library(motifmatchr); library(JASPAR2022)
counts <- SummarizedExperiment(...)   # peak × cell counts
counts <- addGCBias(counts, genome = BSgenome.Hsapiens.UCSC.hg38)
motifs <- getJasparMotifs(species = "Homo sapiens")
motif_ix <- matchMotifs(motifs, counts, genome = "hg38")
dev <- computeDeviations(object = counts, annotations = motif_ix)
vd <- computeVariability(dev)
head(vd[order(-vd$variability), ])
# 1. ATAC-corrected signal
TOBIAS ATACorrect \
  --bam shifted.bam \
  --genome hg38.fa \
  --peaks union_peaks.bed \
  --outdir tobias_out

# 2. footprint scoring
TOBIAS ScoreBigwig \
  --signal tobias_out/sample1_corrected.bw \
  --regions union_peaks.bed \
  --output sample1_footprint.bw

# 3. cross-condition comparison
TOBIAS BINDetect \
  --motifs JASPAR2022_human.jaspar \
  --signals A_footprint.bw B_footprint.bw \
  --genome hg38.fa \
  --peaks union_peaks.bed \
  --outdir bindetect_out
陷阱

常見錯誤

🚫
用整個基因組當 background：HOMER 等若不指定 background，預設用基因組亂數，不公平。建議用 GC-matched 隨機區域或自訂 background peaks。
Using the whole genome as background: HOMER's default uses random genome — unfair. Use GC-matched random regions or custom background peaks.
🚫
Tn5 sequence bias 沒校正：Tn5 對某些序列有 ±1.5× 偏好（如 GC-rich）。Footprinting 前需用 TOBIAS ATACorrect 或 HINT-ATAC 的偏差校正。
Skipping Tn5 sequence-bias correction: Tn5 has ±1.5× preference for some motifs (GC-rich). Correct with TOBIAS ATACorrect or HINT-ATAC before footprinting.
🚫
把 motif enrichment 當作 TF binding 證據：motif 富集只說「這 TF 可能參與」；要確認 binding，需 footprint 或 ChIP-seq。
Treating motif enrichment as binding evidence: enrichment only suggests potential involvement; confirm with footprinting or ChIP-seq.

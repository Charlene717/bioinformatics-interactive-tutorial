ATAC-seq 原理與片段分佈

Tn5 如何只切開放區域、為什麼 fragment size 分佈長得像「鋸齒山脈」。

How Tn5 cuts only open regions and why the fragment-size distribution looks like a "sawtooth mountain".

生化原理

Tn5 轉位酶

Tn5 來自細菌轉位元；工程化版本攜帶 Illumina 接頭。它對 開放、無核小體保護的 DNA 有強烈偏好，會以二聚體形式同時切兩條鏈並將接頭嵌入。一個 ATAC 反應裡，Tn5 完成「切＋接 (tagmentation)」一步搞定。

Tn5 is a bacterial transposase; the engineered ATAC version carries Illumina adapters. It strongly prefers open, nucleosome-free DNA, working as a dimer to cut both strands and insert the adapter. In one ATAC reaction, Tn5 performs "cut + ligate" (tagmentation) in a single step.

切位精度

Tn5 切位形成單鹼基級的訊號。但因蛋白本身會「橋接」9 bp，read 起點對應實際開放邊界要 shift +4 / -5 bp。

Tn5 cuts at single-base precision. Because the protein bridges a 9-bp duplex, read starts must be shifted +4 / -5 bp to match real openness.

片段大小分佈

在 fragment-length 直方圖上會出現三個明顯區段：<100 bp 的 nucleosome-free (NFR)、~180 bp 的 mono-nucleosome、~360 bp 的 di-nucleosome——10.5 bp 的小週期 ripple 來自 DNA 雙螺旋。

The fragment-length histogram shows three peaks: <100 bp nucleosome-free (NFR), ~180 bp mono-nucleosome, ~360 bp di-nucleosome — with ~10.5 bp ripples from the DNA helix.

重要修正

必做的後處理

+4 / -5 shift

Tn5 接合的 9 bp 橋讓 forward read 應 +4、reverse read 應 -5。常用工具：alignmentSieve --ATACshift（deepTools）。

The 9-bp Tn5 bridge means forward reads shift +4 and reverse -5. Tool: alignmentSieve --ATACshift (deepTools).

移除 chrM

粒線體 DNA 完全開放，可佔 30-50% 的 reads。samtools view -F 4 ... grep -v chrM 或專用流程。

Mitochondrial DNA is fully open and can be 30-50% of reads. Use samtools view ... | grep -v chrM or a dedicated pipeline.

移除 blacklist

ENCODE 黑名單包含重複序列與已知偽訊號區。bedtools intersect -v -a ... -b blacklist.bed。

ENCODE blacklist marks repeat/artifact regions. Filter with bedtools intersect -v.

去重

PCR + optical 重複用 picard MarkDuplicates 或 samtools markdup 標記後 -F 1024 過濾。

Mark PCR + optical dups with picard MarkDuplicates or samtools markdup; filter with -F 1024.

程式碼

實作範例

# 1. 比對 (Bowtie2)
bowtie2 --very-sensitive -X 2000 -p 12 \
  -x bowtie2_idx -1 R1.fq.gz -2 R2.fq.gz \
  | samtools view -bS - | samtools sort -o raw.bam

# 2. MarkDuplicates + 過濾 chrM、blacklist、unmapped
samtools markdup -r raw.bam dedup.bam
samtools view -h -b -q 30 -F 1804 -f 2 dedup.bam \
  | grep -v -P 'chrM|chrUn|random' > filt.bam

bedtools intersect -v -a filt.bam \
  -b hg38_blacklist.bed > clean.bam

# 3. Tn5 shift (+4/-5)
alignmentSieve --bam clean.bam -o shifted.bam --ATACshift

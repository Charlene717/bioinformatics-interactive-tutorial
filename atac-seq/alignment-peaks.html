ENCODE 推薦 QC

TSS Enrichment

TSS 周圍 ±2 kb 訊號 / 背景訊號比。> 7 為高品質、5-7 可接受、< 5 低品質。是 ATAC 最重要的單一指標。

Signal at TSS ±2 kb / background. > 7 high quality, 5-7 acceptable, < 5 low. The single most important ATAC metric.

FRiP (Fraction of Reads in Peaks)

落在 peaks 內的 read 比例。ENCODE 建議 > 0.3；超過 0.5 已是優秀。

Read fraction in peaks. ENCODE recommends > 0.3; > 0.5 is excellent.

Library 複雜度

PBC1 = unique positions / mapped positions。> 0.7 良好；過低代表 PCR 過頭或樣本不足。

PBC1 = unique / mapped positions. > 0.7 good; lower indicates PCR over-amplification or too few cells.

Peak 數

大多數人類樣本 ~50,000 - 150,000 peaks (q < 0.05)。過少代表 under-tagmented 或品質差。

Human samples typically 50,000 - 150,000 peaks (q < 0.05). Too few indicates under-tagmentation or poor quality.

完整流程

# MACS3 (or MACS2) 對 ATAC paired-end 推薦設定
macs3 callpeak \
  -f BAMPE -t shifted.bam \
  -g hs -n sample1 \
  --nomodel --shift -75 --extsize 150 \
  -q 0.05 \
  --keep-dup all \
  --outdir macs3_out/
# Genrich 內建 ATAC 模式，無需手動 shift
samtools sort -n shifted.bam -o sorted_qname.bam
Genrich -t sorted_qname.bam \
  -j -y -r -e chrM \
  -o sample1.narrowPeak \
  -k sample1.pileup.bedgraph
# FRiP（落在 peaks 的 read 比例）
in_peak=$(bedtools intersect -u -a shifted.bam -b sample1.narrowPeak | samtools view -c -)
total=$(samtools view -c shifted.bam)
echo "FRiP = $(echo "scale=3; $in_peak / $total" | bc)"

# TSS enrichment（deepTools）
computeMatrix reference-point \
  -S sample1.bw -R tss.bed \
  --referencePoint TSS -a 2000 -b 2000 \
  --outFileName tss_matrix.gz

plotProfile -m tss_matrix.gz -o tss_profile.png \
  --regionsLabel "TSS" --plotTitle "TSS Enrichment"

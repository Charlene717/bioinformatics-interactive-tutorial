核心概念

Germline vs Somatic

🧬

Germline（生殖系）

遺傳自父母的變異，存在於全身細胞。預期 VAF 為 0%、50% (het) 或 100% (hom)。GATK HaplotypeCaller 是標準。

Inherited variants present in every cell. Expected VAF: 0%, 50% (het), or 100% (hom). GATK HaplotypeCaller is standard.

🦠

Somatic（體細胞）

後天發生的變異（癌症突變）。VAF 連續變化（亞群體性），常 < 50% 甚至 < 10%。需 paired tumor-normal 設計與 Mutect2。

Acquired variants (cancer mutations). Continuous VAF (subclonality), often < 50% or even < 10%. Needs paired tumor-normal + Mutect2.

Germline 流程

HaplotypeCaller 流程

對多樣本族群最佳實踐是 GVCF + joint genotyping：每樣本單獨產生 GVCF，再用 GenotypeGVCFs 一起 call。這能讓「在某樣本 ref-ref 但其他樣本有變異的位置」被正確記錄為 ref/ref 而非缺失。

For cohorts, GATK Best Practice is GVCF + joint genotyping: per-sample GVCFs, then GenotypeGVCFs together. Positions that are ref-ref in some samples but variant in others get recorded as ref/ref rather than missing.

# 1. 每樣本生 GVCF
gatk HaplotypeCaller \
  -I patient_01.bam -R GRCh38.fa \
  -ERC GVCF \
  -O patient_01.g.vcf.gz

# 對 WES：加 --intervals 限制目標
# gatk HaplotypeCaller -L exons.bed ...
# 2. 合併 GVCF（族群）
gatk CombineGVCFs \
  -R GRCh38.fa \
  -V patient_01.g.vcf.gz \
  -V patient_02.g.vcf.gz ... \
  -O cohort.g.vcf.gz

# 或用 GenomicsDBImport（大族群更快）
gatk GenomicsDBImport \
  --genomicsdb-workspace-path my_db \
  --sample-name-map cohort.sample_map \
  --intervals chr1

# 3. Joint genotyping
gatk GenotypeGVCFs \
  -R GRCh38.fa -V cohort.g.vcf.gz \
  -O cohort.vcf.gz
# DeepVariant 對 WGS 較簡單
run_deepvariant \
  --model_type=WGS \
  --ref=GRCh38.fa \
  --reads=analysis_ready.bam \
  --output_vcf=patient_01.vcf.gz \
  --output_gvcf=patient_01.g.vcf.gz \
  --num_shards=32
Somatic 流程

Mutect2 paired tumor-normal

# 1. 主呼叫（含 Panel of Normals + germline resource）
gatk Mutect2 \
  -R GRCh38.fa \
  -I tumor.bam -tumor patient01_T \
  -I normal.bam -normal patient01_N \
  --panel-of-normals pon.vcf.gz \
  --germline-resource af-only-gnomad.vcf.gz \
  -O somatic.vcf.gz \
  --f1r2-tar-gz f1r2.tar.gz

# 2. 校正方向偏差
gatk LearnReadOrientationModel \
  -I f1r2.tar.gz -O read-orient-model.tar.gz

# 3. 估計汙染（contamination）
gatk GetPileupSummaries -I tumor.bam -V af-only-gnomad.vcf.gz -L exons.bed -O tumor.pileups.table
gatk CalculateContamination -I tumor.pileups.table -O contamination.table

# 4. 過濾
gatk FilterMutectCalls \
  -V somatic.vcf.gz -R GRCh38.fa \
  --contamination-table contamination.table \
  --ob-priors read-orient-model.tar.gz \
  -O somatic.filtered.vcf.gz

大致原則：Germline（het VAF ~50%）→ 即使 10× 也大致可叫。Somatic VAF 10% → 至少需 30× 才有合理偵測力；VAF 5% 需 ≥ 60-100×。

變異過濾

原始 VCF 含大量 false positives——VQSR 或 hard filter 是必經之路。

Raw VCFs are full of false positives — VQSR or hard filters are mandatory.

兩條路線

VQSR vs Hard Filter

🤖

VQSR

VariantRecalibrator 用已知 truth set (HapMap、1000G、dbSNP) 訓練混合高斯模型，給每個變異一個 tranche-based VQSLOD 分數。需要 ≥ 30 個樣本才會穩定。

VariantRecalibrator trains a Gaussian mixture using known truth sets (HapMap, 1000G, dbSNP); each variant gets a tranche-based VQSLOD score. Needs ≥ 30 samples for stability.

📏

Hard Filter

用固定門檻（QD < 2、FS > 60、MQ < 40 等）篩。小樣本 / 非人類物種 / 沒 truth set 時必用。

Apply fixed thresholds (QD < 2, FS > 60, MQ < 40, etc.). Used for small cohorts, non-human species, or when no truth set is available.

程式碼

實作範例

# 拆 SNP 與 indel 分開 filter
gatk SelectVariants -V cohort.vcf.gz --select-type-to-include SNP -O cohort.snps.vcf.gz

gatk VariantFiltration -V cohort.snps.vcf.gz -O cohort.snps.flt.vcf.gz \
  --filter-expression "QD < 2.0"  --filter-name "QD2"  \
  --filter-expression "FS > 60.0" --filter-name "FS60" \
  --filter-expression "SOR > 3.0" --filter-name "SOR3" \
  --filter-expression "MQ < 40.0" --filter-name "MQ40" \
  --filter-expression "MQRankSum < -12.5"     --filter-name "MQRS" \
  --filter-expression "ReadPosRankSum < -8.0" --filter-name "RPRS"
# 1. 訓練模型 (SNP)
gatk VariantRecalibrator \
  -V cohort.vcf.gz -R GRCh38.fa \
  --resource:hapmap,known=false,training=true,truth=true,prior=15.0 hapmap.vcf.gz \
  --resource:omni,known=false,training=true,truth=false,prior=12.0 1000G_omni.vcf.gz \
  --resource:1000G,known=false,training=true,truth=false,prior=10.0 1000G.phase1.vcf.gz \
  --resource:dbsnp,known=true,training=false,truth=false,prior=2.0  dbsnp.vcf.gz \
  -an QD -an FS -an MQ -an MQRankSum -an SOR -an ReadPosRankSum \
  -mode SNP \
  -O snps.recal --tranches-file snps.tranches

# 2. 套用
gatk ApplyVQSR \
  -V cohort.vcf.gz -R GRCh38.fa \
  -mode SNP \
  --recal-file snps.recal --tranches-file snps.tranches \
  --truth-sensitivity-filter-level 99.7 \
  -O cohort.snps.vqsr.vcf.gz
決策

如何選？

🌳 決策樹

樣本 ≥ 30 + 人類：
VQSR，sensitivity tranche 設 99.7%(SNP) / 99.0%(Indel)。
樣本 < 30 / 非人類 / 無 truth set：
Hard Filter，照官方門檻或社群基準。
CNN：
GATK CNNScoreVariants（小樣本替代 VQSR）可作為單樣本 high-quality filter。
DeepVariant 已輸出：
內建品質分數 (GQ)，不需再跑 VQSR，但仍需做 sample-level QC。
≥ 30 samples + human:
VQSR; tranche 99.7% (SNP) / 99.0% (Indel).
< 30 / non-human / no truth set:
Hard filter with official thresholds.
CNN:
GATK CNNScoreVariants — single-sample alternative to VQSR.
DeepVariant output:
Already includes GQ — VQSR not needed but sample-level QC still required.

變異功能註釋

從「染色體上的座標」躍升到「對基因、蛋白、疾病的潛在影響」。

From "a position on a chromosome" to "potential impact on gene, protein, and disease".

資料庫

必加註釋來源

gnomAD

最大公開人類群體變異頻率資料庫。AF (allele frequency) 是判斷是否為「罕見變異」（< 1% 為罕見）的標準。

Largest public human population AF database. AF distinguishes rare (< 1%) from common variants.

ClinVar

NIH 維護的臨床顯著變異庫。提供 P (pathogenic) / LP / VUS / LB / B 五級分類與審核星等。

NIH-maintained clinical significance DB. Provides P / LP / VUS / LB / B classifications and review-star ratings.

COSMIC

癌症體細胞突變目錄；COSMIC ID、CGC (Cancer Gene Census) 標記重要驅動基因。

Catalog of somatic mutations in cancer; COSMIC ID + CGC (Cancer Gene Census) flags key drivers.

dbSNP / dbNSFP

dbSNP 是已知變異資料庫；dbNSFP 整合 SIFT、PolyPhen、CADD、REVEL 等預測分數。

dbSNP catalogs known variants; dbNSFP aggregates SIFT, PolyPhen, CADD, REVEL prediction scores.

程式碼

實作範例

# VEP 加入 gnomAD AF、CADD、ClinVar 與 LoF (LOFTEE) plugin
vep -i cohort.filtered.vcf.gz \
  -o cohort.vep.vcf.gz --vcf --compress_output bgzip \
  --species homo_sapiens --assembly GRCh38 \
  --fasta GRCh38.fa --cache --offline --dir_cache vep_cache/ \
  --everything --pick \
  --plugin LOFTEE,loftee_path:./loftee,human_ancestor_fa:human_ancestor.fa.gz \
  --plugin CADD,whole_genome_SNVs.tsv.gz,InDels.tsv.gz \
  --custom gnomad.exomes.r2.1.1.sites.vcf.gz,gnomADe,vcf,exact,0,AF \
  --custom clinvar.vcf.gz,ClinVar,vcf,exact,0,CLNSIG
# ANNOVAR：refGene + cytoBand + gnomAD + ClinVar + dbnsfp
table_annovar.pl cohort.flt.vcf humandb/ -buildver hg38 \
  -out cohort.anno \
  -protocol refGene,cytoBand,gnomad30_genome,clinvar_20230520,dbnsfp42a \
  -operation g,r,f,f,f \
  -nastring . -vcfinput \
  -polish
# SnpEff + SnpSift
snpEff -v hg38 cohort.flt.vcf.gz > cohort.snpeff.vcf
SnpSift annotate -info CLNSIG clinvar.vcf.gz cohort.snpeff.vcf > cohort.cs.vcf
SnpSift filter "ANN[*].EFFECT has 'missense_variant' & (CLNSIG = 'Pathogenic')" cohort.cs.vcf
# 快速抽取註釋後欄位 - 適合大量過濾
bcftools +split-vep cohort.vep.vcf.gz -f '%CHROM\t%POS\t%REF\t%ALT\t%Consequence\t%gnomADe_AF\t%CADD_PHRED\t%CLIN_SIG\n' \
  -d -A tab \
  | awk '$6<0.01 && $7>20'   # 罕見 + 高 CADD
臨床優先排序

如何挑出值得驗證的變異？

🌳 優先排序原則

1.
頻率：gnomAD AF < 1%（罕見）或 0（unseen）。
2.
影響：HIGH impact (LoF) 或 MODERATE + 高 CADD (> 20)。
3.
臨床：ClinVar Pathogenic / Likely Pathogenic，且 review status ≥ 2 stars。
4.
基因清單：是否落在 OMIM、HPO 或目標 disease panel 的基因？
5.
遺傳模式：顯性 → het 即可；隱性 → 需 hom 或 compound het；X-linked 等。
1.
Frequency: gnomAD AF < 1% (rare) or 0 (unseen).
2.
Impact: HIGH (LoF) or MODERATE + high CADD (> 20).
3.
Clinical: ClinVar Pathogenic / Likely Pathogenic, review ≥ 2 stars.
4.
Gene list: falls in OMIM / HPO / target disease panel?
5.
Inheritance: dominant → het suffices; recessive → hom or compound het; X-linked; etc.

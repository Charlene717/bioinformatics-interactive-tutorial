技術原理輸入細胞數背景

2025 趨勢：CUT&Tag 取代 ChIP-seq 成為 histone mark 主流；只有 TF 與罕見抗體仍偶用 ChIP。CUT&Tag 與 scCUT&Tag 是單細胞表觀組學的核心。
PIPELINE

② 標準分析流程

FASTQ→ Trim + QC→ Bowtie2 / BWA-MEM→ Filter MAPQ + dedup→ Peak Calling→ IDR / Diff-binding→ Motif / Annotation
  • Peak calling：MACS2/3（narrow：TF / H3K4me3；broad：H3K27me3 / H3K9me3）；SEACR（CUT&Tag/RUN 推薦）。
  • IDR (Irreproducibility Discovery Rate)：ENCODE 標準，至少兩個生物重複才能算可靠 peak。
  • Differential binding：DiffBind（R）、csaw、MAnorm2、THOR。
  • Annotation：ChIPseeker（R）將 peak 連到最近 gene 與基因組區隔（promoter / intron / intergenic）。
  • Motif：HOMER findMotifsGenome.pl、MEME-ChIP、AME。

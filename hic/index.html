Hi-C 與 3D 基因組

DNA 不是一條長線——它在細胞核裡折成 TAD、compartment、loop。3D 結構決定哪些 enhancer 找得到目標基因。

DNA isn't a string — it folds into TADs, compartments, and loops. 3D architecture determines which enhancers reach their target genes.

PRINCIPLE

① Proximity ligation — 在什麼旁邊？

Hi-C 的核心：用甲醛固定染色質→限制酶切→生物素標記接合端→近鄰連接→定序。任何兩段 DNA 出現在同一個 read pair，就代表它們在 3D 空間中很近。把所有 pair 加總，得到 contact map（chr × chr 的對稱矩陣）。

The core idea: formaldehyde-crosslink chromatin → restriction-enzyme cut → biotin-label ends → proximity-ligate → sequence. Any two DNA fragments in the same read-pair were physically close. Summing all pairs yields a contact map — a chr-by-chr symmetric matrix.

③ 標準分析

FASTQ pair→ BWA-MEM / Chromap→ Pair filter (pairtools)→ .mcool / .hic 矩陣→ Compartment / TAD / Loop
  • Tools：HiC-Pro、Chromap（快）、cooler、cooltools、Juicer/JuiceBox（互動視覺化）。
  • Compartment：cooltools eigs-cis / cworld、HOMER runHiCpca.pl（PC1 sign = A/B）。
  • TAD：insulation score（cooltools）、HiCExplorer、TopDom。
  • Loop：HICCUPS（Juicer）、SIP、Mustache、Chromosight。
  • 新平台：Micro-C（更高解析度）、scHi-C（單細胞 3D）、Pore-C（長讀 Hi-C，2024 Nanopore）。

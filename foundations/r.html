③ DESeq2 三行心法

library(DESeq2)
dds <- DESeqDataSetFromMatrix(countData = counts,
                              colData   = coldata,
                              design    = ~ condition)
dds <- DESeq(dds)                          # 估 size factor + 散布 + Wald
res <- results(dds, contrast = c("condition", "treated", "control"))
res <- lfcShrink(dds, coef = 2, type = "apeglm")   # log2FC 收縮（推薦）
summary(res); plotMA(res)

這四行涵蓋整個 bulk RNA-seq 統計核心：raw counts（不要先 normalize）→ NB GLM 擬合 → Wald 檢定 → BH-FDR 校正 → log2FC apeglm 收縮。

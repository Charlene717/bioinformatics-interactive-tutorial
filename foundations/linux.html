Linux 基礎教學

每個 NGS 分析最後都會回到 shell。掌握這 20 個指令，你就能讀懂 95% 的 pipeline。

Every NGS pipeline ends up in the shell. Learn these 20 commands and you'll read 95% of bioinformatics workflows.

WHY

為什麼是 Linux？

幾乎所有體學工具（BWA、samtools、GATK、STAR、Salmon、MACS2）都原生在 Linux 開發；HPC、雲端、Docker container 都是 Linux。Windows / macOS 使用者建議用 WSL2（Windows）或 Homebrew（mac）建立 Linux 環境。

Almost every omics tool (BWA, samtools, GATK, STAR, Salmon, MACS2) is native Linux; HPC, cloud, and Docker containers all run Linux. Windows/macOS users should set up WSL2 or Homebrew.

SURVIVAL KIT

20 個救命指令

# 檔案 / 目錄
pwd                       # 我在哪
ls -lh / ls -la           # 列檔案 + 大小 / 隱藏檔
cd /path/to/dir           # 切目錄；cd - 回上次
mkdir -p a/b/c            # 建多層目錄
cp -r src dst / mv / rm   # 複製 / 搬 / 刪

# 讀檔 — 大檔案救星
head -n 20 file.fastq     # 看前 20 行
tail -f log               # 即時看 log 增長
less file.gz              # 分頁瀏覽（支援壓縮）
zcat file.fastq.gz | head # 壓縮檔流式讀取

# 篩選 / 算 / 改
grep -c "@" reads.fastq   # 算 read 數（@ 開頭）
awk '$5>30' file.sam     # MAPQ>30 過濾
sed 's/chr//' input.bed   # 改字串
cut -f1,3 file.tsv        # 取第 1、3 欄
sort -k1,1 -k2,2n a.bed   # BED 排序（bedtools 需要）
uniq -c                   # 計數

# Pipe & 重導向
samtools view a.bam | awk '$5>30' | wc -l   # 三段串接
cmd 1>out.log 2>err.log                     # 分流 stdout / stderr
cmd &                                       # 背景執行
nohup cmd &                                 # 登出後仍跑

# 程序 / 資源
top / htop                # 看 CPU / 記憶體
ps aux | grep STAR        # 找 process
kill -9 PID               # 強制結束
df -h / du -sh *          # 磁碟用量
PIPELINE PATTERN

一行解決：grep + awk + sort

體學分析有個秘訣：能 pipe 就不要落地中間檔。範例：從 BAM 統計 chr1 上 MAPQ>30 的 read 數：

samtools view -F 4 chip.bam chr1 | awk '$5>30' | wc -l

-F 4 排除 unmapped；$5 是 MAPQ 欄位；wc -l 數 lines。三個小工具串起來就完成原本要寫 50 行 Python 的事。

-F 4 excludes unmapped reads; $5 is the MAPQ column; wc -l counts. Three tiny tools chained do the work of 50 lines of Python.

ENVIRONMENTS

conda / mamba / Docker

# Mamba (conda 的 C++ 加速版，2025 標準)
mamba create -n rnaseq -c bioconda star=2.7 salmon=1.10 multiqc
mamba activate rnaseq

# Docker / Apptainer — 完全可重現
docker run --rm -v $PWD:/data biocontainers/bwa:v0.7.17 \
  bwa mem -t 8 ref.fa R1.fq R2.fq | samtools sort -@4 -o out.bam

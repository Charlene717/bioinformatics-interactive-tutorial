單細胞 RNA 定序

從「組織平均值」走到「每一顆細胞的表達」——細胞類型、狀態、軌跡與整合的核心地圖。

From bulk averages to per-cell expression — the map of cell types, states, trajectories, and integration.

WHY

① 為什麼要做單細胞？

Bulk RNA-seq 把一個組織混在一起測，得到「所有細胞表達的加權平均」。腫瘤、免疫、發育、神經系統的關鍵問題都是「哪一群細胞」在做什麼，bulk 無法回答。scRNA-seq 解析這一層。

Bulk RNA-seq returns the weighted average of all cells in a tissue. The interesting biology of tumours, immunity, development, and the brain is "which subpopulation does what" — a question bulk cannot answer.

PIPELINE

③ 標準流程

FASTQ→ Cell Ranger / STARsolo / Alevin-fry→ QC + Doublet→ Normalize (SCT/log1p)→ PCA → Neighbors → UMAP/Leiden→ Annotation→ DE / Trajectory / Integration
  • 🔍 QC：filter min_genes、max_pct_mt、Scrublet/scDblFinder 偵測 doublet
  • 📊 Normalization：scTransform 或 log1p(CP10K)
  • 🎯 Annotation：marker genes + Azimuth / CellTypist / scANVI
  • 🛤 Trajectory：Monocle3 / PAGA / scVelo
  • 🔗 Integration：Harmony / scVI / scANVI

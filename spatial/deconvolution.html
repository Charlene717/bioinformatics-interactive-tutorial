Deconvolution

把一個多細胞 spot 拆回各細胞類型的比例——讓低解析度平台也能看到細胞級的故事。

Break a multi-cell spot into cell-type proportions — bringing cell-level stories to lower-resolution platforms.

為什麼需要

多細胞 spot 的挑戰

一個 55 μm 的 Visium spot 通常包含 1-10 個細胞，混合多種細胞類型。若直接把 spot 當「一個細胞」處理，會混淆不同 cell type 的訊號。Deconvolution 用單細胞 reference (scRNA-seq) 推估每個 spot 的細胞類型比例。

A 55 μm Visium spot typically holds 1–10 cells of mixed types. Treating a spot as "one cell" mixes signals across types. Deconvolution uses a single-cell reference (scRNA-seq) to estimate the cell-type proportions in each spot.

scRNA reference+ Visium spots→ Deconvolution model→ spot × celltype 比例

程式碼

實作範例

import cell2location as c2l
# 1. 訓練 reference signature
c2l.models.RegressionModel.setup_anndata(adata_ref, labels_key="celltype")
mod = c2l.models.RegressionModel(adata_ref)
mod.train(max_epochs=250)
inf_aver = mod.posterior_mean.cell_type_signatures
# 2. 解構空間樣本
c2l.models.Cell2location.setup_anndata(adata_vis)
m = c2l.models.Cell2location(adata_vis, cell_state_df=inf_aver,
                              N_cells_per_location=5, detection_alpha=20)
m.train(max_epochs=30000)
adata_vis = m.export_posterior(adata_vis)
library(spacexr)
ref <- Reference(sc_counts, sc_cell_types)
puck <- SpatialRNA(coords, sp_counts)
RCTD <- create.RCTD(puck, ref, max_cores = 8)
RCTD <- run.RCTD(RCTD, doublet_mode = "full")
weights <- RCTD@results$weights   # spot × celltype 比例
library(SPOTlight); library(Seurat)
mks <- FindAllMarkers(sc, only.pos = TRUE, min.pct = 0.25)
sp <- SPOTlight(x = sc, y = vis,
              groups = sc$celltype,
              mgs    = mks,
              hvg    = VariableFeatures(sc))
prop <- sp$mat   # 每 spot 的 celltype 比例
陷阱

常見錯誤

🚫
Reference 與 spatial 樣本來自不同組織：用人類肺 scRNA reference 去解構人類腦 spatial，會找不到對應 cell type。理想是同組織、同條件 (FFPE/fresh)、同物種。
Reference and spatial from different tissues: deconvolving brain spatial with lung scRNA fails. Ideal: same tissue, same prep (FFPE/fresh), same species.
🚫
Reference cell-type 註釋過粗：把 「T cell」當一種就無法區分 CD4 / CD8 / Treg。先用精細註釋的 reference。
Coarse reference annotations: labeling all as "T cell" loses CD4 / CD8 / Treg distinctions. Use finely annotated references.
🚫
把 proportions 當 raw counts 做下游 DE：proportions 是 softmax 輸出，不能 DESeq2。要做差異，應在 cluster / 區域層級。
Using proportions as counts for DE: they're softmax outputs — incompatible with DESeq2. Run DE at cluster / region level.

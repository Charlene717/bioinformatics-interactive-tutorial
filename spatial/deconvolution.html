細胞組成解構

把一個多細胞 spot 拆回各細胞類型的比例——讓 Visium 也能講細胞級的故事。

Break a multi-cell spot into cell-type proportions — bring cell-level stories to Visium.

🟢 Beginner🟡 Intermediate🔴 Advanced
開場故事

為什麼一個 spot 是「混合」？

想像你在拍一張人群的衛星空照：解析度只到 50 公尺。你能看到「人群密度」、「移動方向」，但看不到單一個人。Visium 的 spot（55 μm 直徑）就像這張衛星圖——每個點涵蓋 1–10 個細胞，是細胞類型的混合。直接拿來分析會被誤導：你以為某 spot 是「T cell」，其實是 3 個 T cell + 1 個 B cell + 2 個 macrophage 的平均訊號。

Deconvolution 的核心問題：給一個 spot 的表達矩陣，加上同組織的單細胞 reference，能不能反推出每個 spot 中各細胞類型的比例？這是 2020 年後 spatial 領域最熱的方法論競賽，目前主流方法已能達到 RMSE < 10% 的精度。

Imagine a satellite photo of a crowd with 50-m resolution. You see crowd density and movement, but not individual people. A Visium spot (55 μm) is like that satellite image — each covers 1–10 cells and is a mixture of cell types. Naïve analysis misleads: you think a spot is "T cell," but it's really 3 T + 1 B + 2 macrophages averaged.

Deconvolution's core question: given a spot's expression and a single-cell reference of the same tissue, can we recover the cell-type proportions per spot? This has been the hottest methodological race in spatial since 2020; leading tools now achieve RMSE < 10%.

類比：Deconvolution 像「從一杯混合果汁猜出原料比例」——只要你有純的蘋果汁、橘子汁、葡萄汁當參照（reference），就可以反推杯中的混合比例。spatial = 混合杯、scRNA reference = 純果汁庫。
Analogy: Deconvolution is like guessing the proportions in a fruit-juice cocktail — given pure apple, orange, and grape juices as references. Spatial = cocktail; scRNA reference = pure juice library.

三、真實案例：人類腦皮層

Maynard et al. 2021 · Nature Neuroscience · GSE174746

📂 重建 6 層皮層結構

真實資料 Maynard 等人對 12 位健康成人的 dorsolateral prefrontal cortex (DLPFC) 做了 Visium 定序，目的：能否從空間表達無監督地重建大腦皮層的 6 層結構？這是 spatial transcriptomics 經典基準資料集。

方法：(1) 收集同部位 scRNA-seq reference（含 6 層特異的興奮性神經元 marker）；(2) 用 cell2location deconvolve 每個 spot；(3) 把預測比例 overlay 到 H&E 影像上。

結果：不同層別 (L1-L6) 的神經元在 deconv 結果中清楚分佈為平行條帶，與 Nissl 染色一致。白質 (WM) 區域被 oligodendrocyte 比例正確標識。這個資料集現在是 BayesSpace、SpaGCN、cell2location 等工具的「圖靈測試」基準。

啟示：選好 reference 是成功的關鍵——若用 hippocampus scRNA 對 DLPFC 做 deconvolution，這個結構就完全還原不出來。

Real Data Maynard et al. profiled DLPFC (dorsolateral prefrontal cortex) from 12 healthy adults via Visium. Goal: can spatial expression unsupervisedly reconstruct the 6-layer cortical architecture? Now a classic spatial benchmark.

Approach: (1) collect matched scRNA-seq reference (containing layer-specific excitatory neuron markers); (2) deconvolve each spot with cell2location; (3) overlay predicted proportions onto H&E.

Result: L1–L6 neurons appear as clear parallel bands matching Nissl staining. White-matter regions are correctly tagged by high oligodendrocyte proportions. This dataset is now the "Turing test" benchmark for BayesSpace, SpaGCN, cell2location, etc.

Takeaway: reference selection is everything — using hippocampal scRNA on DLPFC would fail to recover this structure.

程式碼

四、實作範例

import cell2location as c2l
# 1. 訓練 reference signature
c2l.models.RegressionModel.setup_anndata(adata_ref, labels_key="celltype")
mod = c2l.models.RegressionModel(adata_ref)
mod.train(max_epochs=250)
inf_aver = mod.posterior_mean.cell_type_signatures
# 2. 解構空間樣本
c2l.models.Cell2location.setup_anndata(adata_vis)
m = c2l.models.Cell2location(adata_vis, cell_state_df=inf_aver,
                              N_cells_per_location=5, detection_alpha=20)
m.train(max_epochs=30000)
adata_vis = m.export_posterior(adata_vis)
library(spacexr)
ref <- Reference(sc_counts, sc_cell_types)
puck <- SpatialRNA(coords, sp_counts)
RCTD <- create.RCTD(puck, ref, max_cores = 8)
RCTD <- run.RCTD(RCTD, doublet_mode = "full")
weights <- RCTD@results$weights
library(SPOTlight); library(Seurat)
mks <- FindAllMarkers(sc, only.pos = TRUE, min.pct = 0.25)
sp <- SPOTlight(x = sc, y = vis,
              groups = sc$celltype,
              mgs    = mks,
              hvg    = VariableFeatures(sc))
prop <- sp$mat

討論

六、思考題

💭 思考點

Q1. 你想分析「腫瘤微環境中 Treg 的空間分布」，但沒有自己的 tumor-specific scRNA reference。可選方案：(a) 用 PBMC 的 Treg signature、(b) 用 Tabula Sapiens 的 Treg、(c) 用其他公開 tumor scRNA。比較三者的優缺點。

Q2. deconv 後一個 spot 顯示「CD4 T = 30% + CD8 T = 30% + Macrophage = 40%」，但同位置 H&E 看起來像純纖維化區。可能原因？提示：reference signature、ambient RNA、deconv 模型限制。

Q3. Visium HD 的 2 μm bin 號稱「single-cell resolution」。是否仍需要 deconvolution？討論：bin 大小 vs 細胞大小、segmentation 演算法、reference 仍有意義嗎？

Q1. You want to map Tregs in the tumor microenvironment but lack a tumor-specific scRNA reference. Options: (a) PBMC Treg signature; (b) Tabula Sapiens Tregs; (c) other public tumor scRNA. Compare pros/cons.

Q2. A spot shows "CD4 T 30% + CD8 T 30% + Macrophage 40%" but the H&E looks like a pure fibrotic region. Possible causes? (Reference signature, ambient RNA, model limitations.)

Q3. Visium HD claims 2 μm bins = "single-cell resolution." Do we still need deconvolution? Discuss bin vs cell size, segmentation, reference utility.

FAQ

七、常見問答

經驗值：每個 cell type 至少 500 cells，total > 10,000 cells。少於這個量，signature 不穩定。Maynard et al. 用 ~78,000 cells 建 reference（DLPFC 約 20 個 cell types）。對稀有 type，可考慮從 multiplex panel 或多筆資料 pooling。

Rule of thumb: ≥ 500 cells per cell type, total > 10,000. Below that, signatures are unstable. Maynard et al. used ~78,000 cells (~20 cell types in DLPFC). For rare types, pool from multiplex panels or multiple datasets.

官方建議 30,000，但實際視訓練 loss 收斂判斷。對 Visium 一個樣本 (~5000 spots) + 20 cell types，30k epochs 在 V100 GPU 約 30 分鐘。若 loss 在 10,000 epochs 已平緩，可早停。

Official recommendation is 30,000, but watch training loss for convergence. For one Visium sample (~5000 spots) + 20 cell types, 30k epochs takes ~30 min on V100 GPU. If loss plateaus by 10,000, early-stop.

Tangram 強項是同時做 gene imputation（把 scRNA 中有但 spatial 沒測到的基因補回 spatial）。若你的研究需要 spatial 的「補完基因表達」，用 Tangram。純做 cell type proportion 估計，cell2location 在 benchmark 上略佔上風。

Tangram's strength is joint gene imputation — recovering scRNA-detected genes that spatial missed. For studies needing "extended gene expression in space," use Tangram. For pure cell-type proportion estimation, cell2location is slightly ahead in benchmarks.

(1) 與 H&E 影像比對：高 immune cell 比例 spots 應對應淋巴浸潤；(2) multiplex IHC 確認關鍵 cell type；(3) 對連續組織 (如皮層) 看是否形成符合解剖的條帶；(4) 跨工具一致性：cell2location 與 RCTD 結果相關係數 > 0.7 才安心。

(1) Compare with H&E: high-immune spots should map to lymphocyte infiltration; (2) confirm key types via multiplex IHC; (3) for continuous tissues (cortex), expect anatomically consistent bands; (4) cross-tool consistency: correlation > 0.7 between cell2location and RCTD = reassuring.

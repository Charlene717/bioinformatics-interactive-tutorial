├─ filtered_feature_bc_matrix.h5    # 主表達矩陣（HDF5）

├─ spatial/
│  ├─ tissue_positions.csv          # spot 像素 / 組織內外

│  └─ aligned_fiducials.jpg
├─ analysis/                         # SpaceRanger 內建分析
└─ web_summary.html                  # 樣本 QC 報告

標準化

空間樣本的標準化

空間樣本標準化大致同 scRNA-seq，但需注意：

Largely the same as scRNA-seq, with a few caveats:

SCTransform

變異穩定的 GLM 殘差，常用於 Visium（Seurat 預設）。對組織異質性區分力強。

Variance-stabilized GLM residuals — common Seurat default for Visium; strong at distinguishing tissue heterogeneity.

LogNormalize

與 scRNA 相同；簡單、跨工具一致。下游有 batch correction（如 Harmony）時表現穩定。

Same as scRNA; simple, cross-tool consistent. Stable when downstream batch correction (Harmony) is applied.

⚠️
勿用空間相關 deconv 結果做 DE：deconvolution 出來的 cell-type 比例不是 raw counts，不能直接餵 DESeq2；要做差異需用對應 cell-level matrices。
Don't DE on deconv-proportions: deconvolution outputs are proportions, not counts — never feed them to DESeq2; use cell-level matrices instead.

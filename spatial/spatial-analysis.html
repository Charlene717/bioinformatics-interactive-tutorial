空間聚類與 SVG

在「不只看表達、也看位置」的前提下，找出組織區域與空間變異基因。

Find tissue domains and spatially variable genes — by leveraging position, not just expression.

核心概念

為何需要 spatial-aware？

標準 Seurat / Scanpy cluster（Louvain on PCA）只看表達相似度。對組織切片來說，這會出現「明明在大腦不同層、表達相似但功能不同」的兩 cluster 混在一起；或同一層被切成多個 cluster。
Spatial-aware clustering 在 cluster 過程同時考量空間鄰近資訊，讓結果更貼近組織形態學。

Standard Seurat / Scanpy clustering (Louvain on PCA) uses expression similarity only. In tissue, that can merge functionally distinct layers with similar profiles, or split one layer into many tiny clusters. Spatial-aware clustering incorporates neighbor information, yielding clusters that match morphology.

SVG

Spatially Variable Genes

SVG 是「表達量隨空間位置有結構性變化」的基因——區隔 HVG（單純表達變異大但無位置模式）。SVG 是空間轉錄組學最關鍵的特徵，下游應用包括：(1) 找組織區域 marker，(2) 找空間 gradient，(3) deconvolution 的訊號基因。

SVGs are genes whose expression varies systematically with location — distinct from HVGs (high variance but no spatial pattern). SVGs are the most important features in spatial data: (1) domain markers, (2) spatial gradients, (3) deconvolution signatures.

Moran's I 直覺

調整空間結構強度，看 Moran's I 變化。I 接近 1 = 強空間結構；接近 0 = 隨機；負值 = 棋盤式分散。

Tune spatial-structure strength; see Moran's I. I ~ 1 = strong structure; ~ 0 = random; negative = checkerboard.

實作範例

library(BayesSpace); library(SpatialExperiment)
sce <- readVisium("sample_outs/")
sce <- spatialPreprocess(sce, n.PCs = 15, n.HVGs = 2000)
sce <- qTune(sce, qs = 3:10)
qPlot(sce)
sce <- spatialCluster(sce, q = 7, nrep = 10000, burn.in = 1000)
clusterPlot(sce)
import NaiveDE, SpatialDE
counts = adata.to_df().T   # gene × spot
norm_expr = NaiveDE.stabilize(counts).T
resid_expr = NaiveDE.regress_out(adata.obs, norm_expr.T, "np.log(total_counts)").T
results = SpatialDE.run(adata.obsm["spatial"], resid_expr)
results.sort_values("qval").head(20)
import squidpy as sq
# Moran's I 計算前需先建立鄰接圖
sq.gr.spatial_neighbors(adata, coord_type="generic")
sq.gr.spatial_autocorr(adata, mode="moran", n_perms=100)

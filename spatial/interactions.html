空間細胞通訊

納入「距離」概念的 ligand-receptor 分析——找出真正在物理上接觸的細胞對話。

Ligand-receptor analysis with distance — find conversations between physically adjacent cells.

核心概念

非空間 vs 空間 LR 分析

scRNA-seq 級的 CellChat、CellPhoneDB 假設「任何兩種 cell type 的 ligand/receptor 共表達 = 可能通訊」——但兩種細胞如果離得遠，paracrine 訊號根本送不到。空間細胞通訊工具在計算 L-R 對交互作用時加入「鄰近權重」，更貼近真實。

scRNA-level tools (CellChat, CellPhoneDB) assume any two cell types co-expressing ligand/receptor = potential communication. But two cells far apart can't realistically send paracrine signals. Spatial CCC tools weight L-R interactions by proximity for more realistic inference.

典型距離尺度：juxtacrine（直接接觸）≈ 1 細胞徑（10–20 μm）；paracrine 訊號 ≈ 50–200 μm；endocrine 跨組織。空間分析中常用 σ ≈ 50–150 μm 對應 paracrine。
Typical scales: juxtacrine (direct contact) ≈ 1 cell diameter (10–20 μm); paracrine ≈ 50–200 μm; endocrine = systemic. σ ≈ 50–150 μm is common for paracrine.
程式碼

實作範例

import commot as ct
# 取得 CellChatDB ligand-receptor pairs
df_LR = ct.pp.ligand_receptor_database(species="human", signaling_type="Secreted Signaling")
# 空間 LR 計算
ct.tl.spatial_communication(adata,
    database_name="cellchat",
    df_ligrec=df_LR,
    dis_thr=500, heteromeric=True)
# 視覺化
ct.pl.plot_cell_communication(adata, k=5)
library(mistyR)
views <- create_initial_view(expr_intra) |>
         add_juxtaview(positions, neighbor.thr = 15) |>
         add_paraview(positions, l = 100)
res <- run_misty(views, results.folder = "misty_out")
misty_results("misty_out") |> plot_view_contributions()
import stlearn as st
st.tl.cci.run(adata, lrs=lr_pairs,
              min_spots=20, distance=None,
              n_pairs=10000, n_cpus=8)
st.pl.lr_summary(adata)
st.pl.cci_check(adata, top_lr="VEGFA_KDR")
驗證

如何驗證 LR 結果可信？

多工具交叉

同時跑 COMMOT、MISTy、stLearn——三者都顯著的 LR pair 最可信。

Run COMMOT, MISTy, stLearn together — LR pairs significant in all three are most trustworthy.

與 scRNA 對照

同一組織也有 scRNA 資料 → 跑 CellChat → 比對哪些 LR 在空間中也出現。

If matched scRNA exists → run CellChat → see which LRs also appear in space.

實驗驗證

最有信心的 LR 對：用 multiplex IHC / RNAscope 在原組織直接看蛋白 / mRNA 共存。

Highest-confidence: confirm with multiplex IHC / RNAscope showing co-localization in the original tissue.

負控制

隨機 shuffle spot 位置後重跑——若仍有顯著 LR，代表結果不是空間驅動的。

Shuffle spot positions and rerun — significant LRs that persist are not spatially driven.

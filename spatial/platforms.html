平台與技術

六大主流空間轉錄組學技術——解析度、通量、價格、適用場景的權衡。

Six major spatial transcriptomics platforms — trade-offs in resolution, throughput, cost, and use cases.

兩大派別

兩大派別

🟢

Array-based

在玻片上固定大量 barcoded 寡核苷酸，捕捉組織釋放的 mRNA，再用 short-read 定序讀取。優：全轉錄組、流程接近 scRNA-seq。限制：解析度受 barcode 大小限制。

Barcoded oligo arrays on a glass slide capture released mRNA; short-read sequencing reads them out. Pro: whole transcriptome, scRNA-like workflow. Limit: resolution capped by barcode size.

代表：Visium、Visium HD、Slide-seq V2、Stereo-seq、HDST。

Tools: Visium, Visium HD, Slide-seq V2, Stereo-seq, HDST.

🟣

Imaging-based

針對預設基因 panel 設計多輪 FISH 探針，逐輪拍照、用組合編碼解讀基因身份。優：亞細胞、保留原位形貌。限制：panel 通常 100-1000 基因，非全轉錄組。

Pre-designed gene-panel FISH probes are imaged in multiple rounds; combinatorial codes decode gene identity. Pro: sub-cellular, preserves morphology. Limit: 100–1000 gene panels, not whole transcriptome.

代表：MERFISH、Xenium、CosMx、STARmap、seqFISH+。

Tools: MERFISH, Xenium, CosMx, STARmap, seqFISH+.

五個關鍵問題

🌳 思考順序

1.
樣本是 FFPE 還是 fresh-frozen？FFPE → Visium FFPE / Xenium / CosMx。
2.
需要全轉錄組嗎？要 → array-based (Visium / Stereo-seq)；不要 → imaging-based 可獲更高解析。
3.
需要單細胞層級嗎？要 → Xenium / MERFISH / Visium HD 或結合 deconvolution。
4.
組織面積多大？大面積（> 1 cm²） → Stereo-seq；標準 → Visium 6.5×6.5 mm。
5.
多模態需求（蛋白、染色）？CosMx / Visium + IF 可同時偵測蛋白。
1.
FFPE or fresh-frozen? FFPE → Visium FFPE / Xenium / CosMx.
2.
Need whole transcriptome? Yes → array-based (Visium / Stereo-seq); no → imaging-based for higher resolution.
3.
Need single-cell? Yes → Xenium / MERFISH / Visium HD or pair with deconvolution.
4.
Large tissue (> 1 cm²)? → Stereo-seq; standard → Visium 6.5×6.5 mm.
5.
Multimodal (protein, IF)? CosMx / Visium + IF for simultaneous protein detection.

Profile genome, expression, chromatin, and proteins on the same samples — and stitch the fragments back into a complete story.

METHODS

② 五大方法家族

  • Latent variable / matrix factorisation：MOFA / MOFA+（Argelaguet 2018, 2020）、iCluster、JIVE — 找跨組學共同 latent factors。
  • Canonical correlation：DIABLO（mixOmics）、sparse CCA — 找最大相關的線性組合。
  • Network-based：SNF (Similarity Network Fusion)、WGCNA per omic + cross-module correlation。
  • Variational autoencoder / deep：totalVI (RNA + 抗體)、MultiVI (RNA + ATAC)、scMM、Cobolt、Mowgli。
  • Optimal transport：UnionCom、SCOT、Pamona — 對未配對樣本做跨模態對齊。
PITFALLS

③ 三個常見地雷

⚠️
① Batch confounding：不同 omics 的 batch 必須先個別校正後再整合，不要丟給整合演算法處理。
② 不同尺度的 missing：WGS 幾乎沒 missing，但 proteomics 與 single-cell ATAC missing 比例可達 30-90%；不能用同一個 imputation 策略。
③ 「整合等於全部一樣」：整合後的 UMAP 看起來分開不一定代表分群錯——某些 modality 本來就無法區分某些細胞類型（如 chromatin accessibility 無法解析 myeloid 子群）。
PATTERN

④ Multiome 範例：RNA + ATAC

# scvi-tools MultiVI 整合 RNA + ATAC (10x Multiome)
import scvi
scvi.model.MULTIVI.setup_anndata(adata_multi, batch_key='batch')
model = scvi.model.MULTIVI(adata_multi,
                           n_genes=(adata_multi.var['modality']=='gex').sum(),
                           n_regions=(adata_multi.var['modality']=='atac').sum())
model.train()
adata_multi.obsm['X_multivi'] = model.get_latent_representation()
# 用 latent 做 UMAP / Leiden — 同時利用兩個 modality 的資訊
